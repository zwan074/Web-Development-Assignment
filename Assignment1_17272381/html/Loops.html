Javascript Loops

It is handy to use Loops to iterate functions

JavaScript supports different kinds of loops:

  • for - loops through a block of code a number of times
  • for/in - loops through the items of an object
  • while - loops through a block of code while a specified condition is true
  • do/while - another format of while loop

The "For" Loop.

The for loop has the following syntax:

for ( statement 1; statement 2; statement 3) {
code block to be executed
}
  • statement 1 is executed (one time) before the execution of the code block.
  • statement 2 defines the check condition for executing the code block.
  • statement 3 is executed (every time) after the code block has been executed.
let i;
for (i = 0; i < 5; i++) {
alert(i) ;
}

From the example above, you can read:

Statement 1 sets a variable before the loop starts (let i = 0).

Statement 2 defines the condition for the loop to run (i must be less than 5).

Statement 3 increases a value (i++) each time the code block in the loop has been executed.

The For/In Loop

The JavaScript for/in statement loops through the items of an object:

let student = {firstName:"Jim", lastName:"Dee", age:20};

let x;
for (x in student) {
alert(student[x]);
}


The "While" Loop

The while loop loops through a block of code as long as a specified condition is true.

while (condition) {
code block to be executed
}

Example:

In the following example, the code in the loop will run if i is less than 10,:

let i = 0;
while (i < 10) {
alert( "The number is " + i);
i++;
}


The "Do/While" Loop

The do/while loop is another version of the while loop. This loop will execute the code block in "do" block once,
before checking if the condition is in "while" block true

do {
code block to be executed
}
while (condition);

Example

let i = 0;
do {
alert("The number is " + i);
i++;
}
while (i < 10);

It needs to declare a halt condition, otherwise the loop will run forever!


Comparing For and While

For and While loop could be transferred each other by adjusting their formats

The loop in this example uses a for loop to collect the student names from the students array:

let students = ["Bob", "Vin", "Sam", "Fen"];
let i = 0;
for (;students[i];) {
alert ( students[i]);
i++;
}

The loop in this example uses a while loop to collect the car names from the cars array:

let students = ["Bob", "Vin", "Sam", "Fen"];
let i = 0;
while (students[i]) {
alert(students[i]) ;
i++;
}

Summary

3 types of loops have been covered in this section:

  • for (;;) – The condition is checked before each iteration, additional settings available.
  • while – The condition is checked before each iteration.
  • do..while – The condition is checked after each iteration.

Usually the while(true) construct an “infinite” loop. It can be stopped with the break in such loop.

We can pass current content by using the continue .

Use the for loop to generate even numbers from 2 to 11.

for (let i = 2; i <= 11; i++) {
if (i % 2 == 0) {

}
}

We use the “modulo” operator % to get the remainder and check for the evenness here.
