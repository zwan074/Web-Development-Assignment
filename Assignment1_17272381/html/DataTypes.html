Javascript Data types

A variable in JavaScript can contain any type of data. It can also be reassign to other type of data:

let message = "hello";

message = 257897897 ;

We call such things as “dynamically typed”, we’ll study the basics in this section.

Number

let n = 321;

n = 321.345;

The number type has both for integer and floating point numbers.

There are many operand for numbers, e.g. multiplication * , division / , addition + , subtraction - and so on.

Apart from regular numbers, there are so-called “special numeric values” which also belong to that type: Infinity , -Infinity and NaN .

Infinity represents the mathematical Infinity ∞.

alert( 1 / 0 ); // Infinity

Or just mention it in the code directly:

alert( Infinity ); // Infinity

NaN represents a computational error. It is a result of an incorrect or an undefined mathematical operation, for example:

alert( "not a number" / 2 ); // NaN, wrong type of data to operate

NaN is sticky. Any further operation on NaN would give NaN:

alert("not a number" / 2 + 5 ); // NaN

So, if there’s NaN in a expression/code block, it continuously pass to the whole result.

String

String in JavaScript must has a quote.

let str = "Hello";

let str2 = 'Single quotes are ok too';

let phrase = `can embed ${str}`;

In JavaScript, there are 3 types of quotes.

  1. Double quotes: "Hello" .

  2. Single quotes: 'Hello' .

  3. Backticks: `Hello` .

Double and single quotes are “simple” quotes. There’s no difference between them in JavaScript.

Backticks are “extended functionality” quotes. They allow us to embed variables and expressions into a string by syntax as ${…} , for example:

let name = "John";

// embed a variable below

 alert( `Hello, ${name}!` ); // Hello, John!

// embed an expression

 alert( `the result is ${1 + 2}` ); // 3

The expression inside ${…} is evaluated and the result becomes a part of the string.

Please note that this can only be done in backticks. Not working in other type of quotes

alert( "the result is ${1 + 2}" );

// the result is ${1 + 2} (not working double quotes)

Boolean (logical type)

The foundation of computer only has two states 0 and 1.Therefore, The boolean type has : true and false .

This type is commonly used to store yes/no values: true means “yes, correct, or 1”, and false means “no, incorrect, or 0”.

For instance:

let studentNameFilled = true; // yes, student name has been filled
let studentAgeFilled = false; // no,student age field has been filled

Boolean values can also be used as a result for comparisons:

let isGreaterThan = 4 > 1;

alert( isGreaterThan ); // true (the comparison result is "yes", since 1 means true)

Summary

  • number has integer or floating-point.
  • string may have one or more characters.
  • boolean for true /false .
