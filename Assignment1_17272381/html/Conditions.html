Javascript Conditions

Condtion Statement

You can use conditional statements in your code to perform different actions for different decisions.. In JavaScript we have the following conditional statements:

  • Use if to decide a block of code to be executed, if a specified condition is true
  • Use else to decide a block of code to be executed, if the same condition is false
  • Use else if to decide a new condition to test, if the first condition is false
  • Use switch to decide many alternative blocks of code to be executed

The if Statement

Use the if statement to specify a block of JavaScript code to be executed if a condition is true.

For example:

let year = prompt('In which year was google founded', '');

if (year == 1998) alert( 'Correct!' );

The else Statement

Use the else statement to specify a block of code to be executed if the condition is false.

For example:

let year = prompt( 'In which year was google founded', '');

if (year == 1998) {
alert( 'Corrct!' );
} else {
alert( 'Please do some research?' ); // any value except 1998
}

The else if Statement

Use the else if statement to specify a new condition if the first condition is false.

For example:

let year = prompt( 'In which year was google founded', '');

if (year 1998) {
alert( 'A little bit early...' );
} else if (year > 1998) {
alert( 'A little late' );
} else {
alert( 'Correct!' );
}

1.if (a string has 1)

Will alert be shown?

if ("1") { alert('Hi' ); }

Yes, it will. Any string except an empty one (and "1" is not empty) will be read as true in the boolean context.

We can run and check:

if ("1") {

alert('Hi' );

}
